Changee Learning Path - 網站製作實務

在網路的時代，我們在開始學習新領域時，可以透過 Google 找到許多學習的「資訊」，然而過多的資訊反而無法判斷什麼才是適合自己的「資源」。現有的學習資源非常豐富，但卻散落在各處，使得「自我學習」時總是花費大量時間在搜尋適合自己的資訊。因此我們希望能夠建立 Changee Learning Path「學習路徑平台」，整合資源與資訊，透過「學習分類帽」、「學習軌跡」、「實體資源」三個主要項目，建立方便自學的「學習路徑平台」。

Webfont

受夠了內建字體嗎？

Google fonts

  • 不用擔心使用者電腦內沒有安裝字型
  • Open Source 免費
  • Serif、 Sans-serif、 Display、 Handwriting、Monospace 任君挑選
  • 多樣的引入方式

justfont 中文字型服務

  • 設計師受夠了新細明體
  • 亞洲語系字型檔動輒 5~6MB 載入時間太久
  • 依據您網頁上使用的文字，動態的產生並下載您需要的字型

身為前端工程師，字型也要稍微了解

  • justfont 部落格
  • 字嗨社團
  • Type is Beautiful

icon 服務

iconion

  • icon 產生器
  • 陰影漸層調整
  • 現成 icon 選擇，不重造輪子

Font Awesome

  • 之前介紹過
  • 向量文字，放大解析度還是一樣
  • One Font, 439 Icons 載入許多用不到的 icon

fontastic

  • 顏色、大小、漸層、陰影都由 CSS 處理
  • 客製化需求，不引入多餘 icon

fontello

  • 支援 SVG 轉換成 icon font

icomoon

  • 支援 icon 輸出成 PNG, SVG
